# TOML Editor

一个基于 Vue 3 + Vite 的可视化 TOML 配置文件编辑器。

## 功能特性

- 🎨 可视化编辑界面
- 📝 实时 TOML 预览
- 💾 配置文件上传/下载
- 🔄 本地存储支持
- 🎯 语法高亮显示

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages。

推送到 `main` 分支后会自动触发部署流程。

## 技术栈

- Vue 3
- Vite
- TailwindCSS
